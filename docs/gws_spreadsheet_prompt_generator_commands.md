# GWS CLI build commands — Spreadsheet Prompt Generator

Use these commands in order to build a Google Sheet named **Spreadsheet Prompt Generator**.

> Notes:
> - Commands are written as one-line Windows Command Prompt compatible commands.
> - JSON bodies are escaped for CMD using double quotes.
> - `sheetId` mapping used throughout:
>   - `0` = `Prompt Generator`
>   - `1` = `Examples`
>   - `2` = `Reference` (hidden)

## Step 1 — Create spreadsheet

```cmd
gws sheets spreadsheets create --data "{\"properties\":{\"title\":\"Spreadsheet Prompt Generator\"},\"sheets\":[{\"properties\":{\"sheetId\":0,\"title\":\"Prompt Generator\",\"gridProperties\":{\"rowCount\":300,\"columnCount\":12}}},{\"properties\":{\"sheetId\":1,\"title\":\"Examples\",\"gridProperties\":{\"rowCount\":220,\"columnCount\":10}}},{\"properties\":{\"sheetId\":2,\"title\":\"Reference\",\"hidden\":true,\"gridProperties\":{\"rowCount\":300,\"columnCount\":12}}}]}"
```

Save the returned spreadsheet ID as `%SID%`.

```cmd
set SID=PASTE_SPREADSHEET_ID_HERE
```
